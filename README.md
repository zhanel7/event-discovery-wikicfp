Ғылыми конференциялар мен симпозиумдарды іздеуге арналған веб-қосымша. Бұл жоба WikiCFP сияқты сервистердің үлгісі бойынша жасалған және пайдаланушыларға конференцияларды қарау, іздеу және әкімшілерге оларды басқару мүмкіндігін береді.


- Пайдаланушы аутентификациясы (тіркелу, кіру, шығу)
- Рөлдер: қарапайым пайдаланушы және әкімші
- Конференциялармен CRUD операциялары (әкімшілер үшін)
- Конференциялар тізімі (барлық пайдаланушылар үшін)
- Іздеу және сүзгілеу
- RESTful API
- Адаптивті интерфейс (Bootstrap)
- Әкімшілік панель (Django admin)
- API құжаттамасы (Swagger UI)


- **Бэкенд:** Django, Django REST Framework
- **Дерекқор:** PostgreSQL
- **Фронтенд:** HTML, Bootstrap
- **Контейнеризация:** Docker, docker-compose
- **CI/CD:** GitHub Actions
- **Деплой:** Render
- **Мониторинг:** Prometheus, Grafana
- **Жүктеме тестілеу:** k6
